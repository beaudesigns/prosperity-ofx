# Prosperity OFX

> An open source OFX/QFX parser

## Installation

```
npm install prosperity-ofx
```

## Running Tests

```
npm test
```

## API

#### .parse()
Convert OFX or QFX data to a JavaScript object literal.

#### .encode()
Convert your JavaScript Object to OFX.

## Supported OFX Specs

- [x] 1.0.2 specification
- [x] 1.0.3 specification
- [ ] 2.0.3 specification
- [ ] 2.1.1 specification
